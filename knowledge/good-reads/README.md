## What This Is

Articles, essays, and blog posts I've read and want to keep. Not everything I read — just the ones that stuck. The ones I'd recommend to someone building a company or trying to think more clearly.

Each article is saved as a full markdown file (scraped via Jina), with metadata for author, publication, tags, and why it resonated.

**Why save the full text?**
- Links rot. Substacks get paywalled. Blogs go down.
- Full-text search across everything I've read.
- AI can reference these when I'm working on related problems.

---

## How to Add

Use the **save-read** skill:

```
Save read: https://www.danhock.co/p/becoming-an-artist
```

The skill scrapes the page, formats it as markdown with metadata, names it properly, updates this README and INDEX, and adds the author/publication to `sources/`.
