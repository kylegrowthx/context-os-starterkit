## Context

Augment Code is facing a critical inflection point: while organic traffic is recovering incrementally (3% WoW), the trajectory suggests only 50k page views/week by year-end if current tactics continue—insufficient to achieve growth targets. Liz presented successful models from two clients (DataGrid and Webflow) where programmatic SEO engines around technical use cases and integrations are driving measurable results. DataGrid is pivoting to product-led discovery via agent-based content clusters, while Webflow has scaled integration guides to 30/week with strong SEO performance. For Augment Code, the insight is clear: developer search intent (Google and AI) remains consistent around "how do I build X with Y tools." The opportunity is to author at scale using AI, embed product CTAs (both traditional copy prompts and deep links to the new Intent app), and track performance through both traffic metrics and AI visibility via CheckThat. This approach transforms content from a passive inbound channel into an active conversion funnel.

---

## Relevance

- **Programmatic SEO / Content Scale:** Shift from incremental guide production to AI-driven, parametric content generation at scale (100s of guides) across technical stacks and use cases.
- **Product-Content Loops:** Embed Intent app CTAs (both narrative copy and deep links) into how-to guides; developers reading "How to build a blog with Next.js + Sanity" should be shown the app as the fastest execution path.
- **AI Visibility & CheckThat:** Track PSEO content performance not just in Google Search Console but in AI model recommendations (e.g., does Augment show up when GPT users ask about code-generation workflows?).
- **Competitive Keywords:** Aggressively own "Cursor alternative" and high-intent comparison keywords; reduce reliance on expensive paid ad spend by winning organic share.
- **Content Quality at Scale:** Validate that AI-generated technical content (Next.js v16, Sanity v3 blogs, etc.) with good system prompts produces copy-pastable, functional code without extensive QA.
- **Cross-Client Learning:** Apply lessons from DataGrid (agent-based clusters) and Webflow (integration pages + code components) to Augment Code's own developer audience.

## Key Topics

### Current Performance & SEO Health

  - Organic traffic is recovering from a November dip (from 40k to 15k page views/week) with steady \~3% WoW growth.
  - **Technical Issues:**
      - Broken redirects were identified → a fix ticket is being created.
      - A canonical page is needed to resolve a ranking conflict between "cursor" and "augment" keywords.
  - **AEO:** The AEO workstream is on hold pending a clearer strategy.

### New Strategy: Programmatic SEO (PSEO)

  - **Goal:** Create a "big bet" to accelerate growth beyond the current incremental pace.
  - **Model:** A programmatic content engine for technical "how-to" guides, inspired by GrowthX's work with DataGrid and Webflow.
  - **Content Focus:**
      - Highly technical, solution-oriented guides (e.g., "How to build a blog with Next.js & Sanity").
      - Tool comparisons and "best X for Y" lists.
  - **Rationale:** This content directly answers developer search intent, which is consistent across traditional search and AI prompts.

### Conversion Hypothesis: Content → App

  - **Core Idea:** Convert high-intent users from PSEO content into app users.
  - **Mechanism:** Embed deep-link prompts in guides that, when clicked, download the Intent app with the tutorial's code pre-loaded.
  - **Narrative:** Position the app as the most efficient way to execute the tutorial's workflow, building trust by demonstrating value upfront.

### Competitive Strategy: Cursor Alternative

  - **Opportunity:** Target "Cursor alternative" keywords to capture high-intent users actively seeking a switch.
  - **Rationale:** Success here could reduce paid ad spend on competitive keywords, freeing up budget for the new PSEO initiative.
