# Brex

Brex is a fintech spend management platform (acquired by Capital One, Jan 2026). GrowthX is their SEO and content growth partner, focused on tools, indexes, and templates to hit aggressive lead targets.

## Directory Structure

```
brex/
├── brex-client-context-v1.md        # Start here — relationship, engagement history, file map
├── context/                          # Brand guidelines, personas, reference docs
│   ├── engagement-snapshots/         # Point-in-time snapshots after key calls or phase changes
│   └── links.md                      # External links (Drive, Notion, Figma)
├── deliverables/                     # Work product organized by workstream
│   ├── web-experience/               # Resources page redesign (current primary focus)
│   ├── strategy/                     # Overall engagement strategy docs
│   └── experimental-lane/            # Indexes + templates
├── transcripts/                      # Meeting transcripts
├── resources/                        # Competitive research data (CSVs, exports)
└── examples/                         # Reference examples of their content
```

## How to Use

1. **Always start with `brex-client-context-v1.md`** — it has the relationship context, engagement history, and a file map that tells you which files to load for what purpose.
2. Load additional files from the file map based on the task at hand.
3. Search transcripts for specific topics rather than loading them whole.
