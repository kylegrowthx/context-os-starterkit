# Vercel

Vercel is the leading cloud platform for frontend and full-stack web development. Creators of Next.js, v0, AI SDK, and AI Gateway. $9.3B valuation, $200M+ ARR. GrowthX is their editorial content and programmatic SEO partner, focused on AEO/GEO, content velocity on vercel.com/i, and programmatic stacks pages.

## Directory Structure


## How to Use

1. **Always start with `vercel-client-context-v1.md`** — it has the relationship context, engagement history, and a file map that tells you which files to load for what purpose.
2. **For content work**, also load `context/guillermo-rauch-founder-deep-dive-v1.md` — it contains the definitive voice and tone analysis for writing that feels "Vercel."
3. Load additional files from the file map based on the task at hand.
