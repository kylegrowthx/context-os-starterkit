## Overview

- WordPress is being used as the CMS, with the RankMath plugin for SEO optimization
- Categories are used to create topic clusters and structure URLs
- Featured images should be optimized for size and resolution
- RankMath provides SEO analysis, meta data customization, and social preview options
- AI content generation tools are available but need to be tested

---

## Key Topics

### WordPress Overview

  - Posts vs Pages: Posts are for blog/dated content, Pages are for static website content
  - Customer success cases are separated into their own custom post type
  - Existing category structure needs improvement to align with topic clusters

### Post Editing

  - Standard editor for text, images, formatting
  - Elementor plugin available for more complex layouts if needed
  - Post settings: status, visibility, author, categories, tags, featured image
  - Categories form the URL path, should be carefully structured

### SEO with RankMath

  - Previews meta title, description, social sharing cards
  - Analyzes content for focus keywords, readability, etc.
  - Allows customizing schema markup
  - Has AI content tools that need testing
