## Key Topics

### Current SEMrush Usage & Transition

  - GrowthX currently has 47 users on monthly core plan
  - Using SEMrush API in existing workflow; critical to maintain API key continuity
  - Transitioning to Enterprise requires matching core and enterprise user counts
  - Guest users have limited access (can view reports but not execute workflows)

### User Access Requirements

  - 11-12 director-level users need full enterprise access
  - Additional 12-13 managing editors may need access (total 24-25 users)
  - Remaining users could potentially be downgraded to guest status
  - Guest users can view reports but cannot action items in core product

### Pricing and Contract Terms

  - Requested pricing for both 12 and 25 user options
  - Target budget: $7,500-$8,000 per month
  - Enterprise plans typically don't offer user-based discounts
  - Payment terms: Quarterly payments required (no monthly option for Enterprise)
  - Payment method: Wire transfer or ACH only (no credit cards for Enterprise)

### Contract Details

  - Annual commitment required for Enterprise plan
  - Silver package includes: 100 projects, 10,000 keywords, 20,000 page speed insights, 500 monthly service credits, 2 million API units
  - Includes Google Search Console and GA4 integrations
