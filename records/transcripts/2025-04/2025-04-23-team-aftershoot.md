## Overview

- AfterShoot dramatically speeds up photo culling, editing, and retouching for photographers
- The AI learns user preferences and can be trained on custom editing styles
- Workflow: import → AI culling/editing → review in AfterShoot → export to Lightroom for final tweaks
- Retouching features allow quick adjustments to skin, clothing, etc. without manual editing

---

## Key Topics

### AfterShoot Workflow Overview

  - Import photos directly from SD cards or external drives
  - Select genre (e.g. wedding) for AI tailored to that style
  - One-click culling and editing option processes overnight
  - Review AI selections and edits in the morning
  - Export to Lightroom for final adjustments

### AI Culling Features

  - Selects best shots from similar groupings
  - Filters for technical issues (e.g. closed eyes)
  - Dramatically reduces number of photos to review (e.g. 12,000 → 1,500)
  - User can easily add/swap selections

### AI Editing Capabilities

  - Uses custom user profile trained on 2,500-10,000+ edited images
  - Applies user's editing style across all photos dynamically
  - Handles various lighting conditions based on training
  - Marketplace profiles available for different styles

### Retouching Tools

  - Sliders for skin issues, wrinkles, clothing adjustments, etc.
  - Currently requires exporting to/from Lightroom as JPEGs
  - Working on integrating directly into main workflow

### Time Savings

  - Reduces edit time from weeks to days for large events
  - Example: 12,000 wedding photos culled/edited overnight
  - Allows photographers to focus on shooting, not editing
