## Key Topics

### Roadway Platform Overview

  - Centralized attribution management hub for parsing sessions and applying custom rules
  - Connects various data sources (e.g., CRM, analytics tools) to provide a holistic view of the customer journey
  - Offers flexible attribution models (first-touch, multi-touch, last-touch) with customizable rules and weights
  - Provides campaign-level insights on revenue metrics, including real ROAS calculations

### Technical Capabilities

  - Uses a pixel for website tracking and integrates with ad platforms for spend data
  - Allows for CRM overrides to include offline touchpoints and custom events
  - Offers retroactive attribution updates when new rules or sources are added
  - Provides a semantic layer for easy dashboard creation and analysis

### AI-Powered Analysis

  - "Growth marketing coworker" feature automatically analyzes trends and surfaces insights
  - Ability to create custom "coworkers" focused on specific areas (e.g., SEO) or metrics
  - Generates action reports for each metric, considering full-funnel impact and validation metrics

### Agency Partnership Model

  - Roadway offers a 25% discount to agency clients and a 25% commission to the agency
  - Agencies can add service margins on top of the Roadway subscription
  - Roadway provides support in client onboarding, including attribution mapping and dashboard setup

### Implementation Process

  - Client connects data sources through OAuth or API keys
  - Roadway team validates data and conducts a kickoff call to understand attribution needs
  - Custom workspace with tailored dashboards delivered within 3 days
  - Ongoing support for tweaks and optimizations
