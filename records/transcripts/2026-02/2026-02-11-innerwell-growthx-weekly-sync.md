## Key Topics

### Insurance Hub Strategy

  - **Problem:** The current insurance directory is buried in the help section, failing to answer the most common member question and creating a conversion barrier.
  - **Goal:** Create a prominent insurance hub in the main navigation to provide clear, upfront information.
  - **Proposed Structure:**
      - **Primary Page:** A central "Coverage" or "Insurance" page listing all accepted providers.
      - **Sub-Pages:** Individual pages for each insurer, potentially organized by state.
  - **Key Feature:** An on-page eligibility check (like Talkiatry's) that funnels users directly into the assessment, shortening the conversion path.

### Insurance Content Cluster

  - **Goal:** Build an insurance-focused content cluster to capture high-intent search traffic.
  - **Strategy:** Replicate the success of the "Ketamine Laws by State" article (\~1k sessions) by creating a "Ketamine Insurance by State" series.
  - **Blog Updates:** Update \~20 existing articles with stronger insurance mentions and CTAs. This will be prioritized after the new hub is built.
  - **Resource Request:** GrowthX needs more detailed insurance documentation from Innerwell to create deep, authoritative content.

### Medication Content Strategy

  - **Early Success:** The new "Alternatives" pages are performing well (e.g., Prozac Alternatives: 81 sessions), validating the strategy.
  - **Future Direction:** Apply the insurance hub model to medications, creating a "Medications" tab with dedicated pages.
  - **New Format Idea:** Create visual comparison pages (e.g., "Innerwell vs. Prozac") as a more engaging funnel step than a standard blog.
