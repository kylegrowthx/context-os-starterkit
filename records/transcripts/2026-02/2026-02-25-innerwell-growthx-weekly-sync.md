## Key Topics

### Ketamine Page Optimization

  - **Problem:** Paid ad coverage for high-intent keywords ("ketamine treatment online") has dropped since Nov/Dec. The main Ketamine page ranks poorly for these terms, while competitors' main service pages rank highly.
  - **Solution:** Optimize the main Ketamine page (`/ketamine`) to improve organic ranking and capture high-intent traffic.
      - **Actions:** Update copy, expand on insurance, improve internal linking, and refresh metadata.
      - **Rationale:** The homepage cannot be fully optimized for Ketamine without skewing the site's multi-service focus, making the dedicated service page critical for ranking.

### Blog Traffic Drop Investigation

  - **Problem:** A significant traffic drop on blog pages began around Feb 17.
  - **Hypothesis:** The timing coincides with a banner UTM update by Shinali on Feb 17.
  - **Action:** Matt will contact Shinali for details to provide to the analytics team for investigation.

### New Index Pages (Psychiatry & Therapy)

  - **Scope:** New index pages for Psychiatry and Therapy will use the same 33-state provider list as Ketamine. EMDR is separate with 17 states.
  - **Status:** In design; targeting early next week completion.
  - **Bug:** Filtering by service on a city page redirects users to the state page. Matt will report this to Ilan.
  - **Enhancement:** Add a "Can't find your city?" link to dropdowns, directing users to the state page. Matt will consult the designer on UX.

### Insurance Landing Page

  - **Goal:** Create a central hub for all insurance information, replacing the current screener CTA.
  - **Plan:**
      - **GrowthX:** Provide layout structure and copy.
      - **Innerwell:** Tomasov & Ilya will build the page.
      - **Both:** QA the final page.

### Alcohol Use Disorder (AUD) Page

  - **Status:** AUD service is still in testing but showing promising purchase data.
  - **Future Plan:** If launched, create a dedicated AUD service page and add it to the main navigation.
  - **Priority:** This project is now secondary to Ketamine page optimization.

### Editorial Strategy & Content

  - **Psychiatry Content:** A content roadmap is in progress, mirroring the successful Ketamine strategy (awareness, access, comparisons). The "comparisons" cluster needs further definition (e.g., comparisons vs. competitors or vs. different medications).
  - **Production Cadence:** Publish 1–2 Psychiatry-focused articles per week, increasing from current rate.
  - **Testimonial CTAs:** Matt proposed adding video testimonials as banner breaks in blog content to shallow the funnel (e.g., link to a review page instead of a screener). This approach successfully used for other B2B clients.
      - **Action:** Akina will provide available testimonials for Matt to review.
