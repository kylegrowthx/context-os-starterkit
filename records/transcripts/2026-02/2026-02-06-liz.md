## Key Topics

### Agent Strategy: Existing vs. Custom

  - **Problem:** A strategic disconnect exists between Liz's one-pager (focusing on existing agents) and Marcel's vision of custom, "spin-up" agents.
  - **Solution:** Adopt a phased approach to build a stable process before expanding.
      - **Phase 1:** Focus on content for the existing suite of agents.
      - **Phase 2:** Tackle custom, spin-up agents later, likely led by Nico.

### Content Structure: Guides vs. PSEO

  - **Problem:** The new content architecture risks cannibalizing keywords and creating user confusion with existing blog posts.
  - **Solution:** Define clear roles for new content types.
      - **Guides (TOFU):** High-level educational content (e.g., "What is an RFI in Construction?").
          - Minimal agent promotion → simple CTAs only.
      - **PSEO Pages (BOFU):** Specific use cases tied to agents (e.g., "How to Use the RFI Agent to Screen Submittals").
          - Detailed agent functionality and workflows.
  - **Decision:** Refresh existing blog posts as new guides to preserve SEO history and avoid creating duplicate content.

### Content Generation: Top-Down vs. Bottom-Up

  - **Problem:** The best method for generating a comprehensive list of specific use cases is unclear.
  - **Two Approaches:**
      - **Top-Down (Persona-Based):**
          - Start with industry personas and their jobs-to-be-done.
          - Generate seed keywords → manual research → shoehorn in agents.
          - *[Likely to produce more general guide topics.*](https://fathom.video/share/xv3FvGQ1fckD9YrnQxBzsnDGAAhKCHqm?tab=summary&timestamp=1219.0)
      - **Bottom-Up (Agent-Based):**
          - Start with an agent and list all its capabilities per industry.
          - *[Likely to produce more specific PSEO use cases.*](https://fathom.video/share/xv3FvGQ1fckD9YrnQxBzsnDGAAhKCHqm?tab=summary&timestamp=1310.0)
  - **Decision:** Test the bottom-up approach on two agents to evaluate its effectiveness.

### PSEO Page Templates

  - **Problem:** The "best agent for X" PSEO concept is flawed because users create their own ideal agents on the platform.
  - **Decision:** Liz and Talal will research effective PSEO page templates for this unique product.

### Newsletter Status

  - **Problem:** The newsletter's value is unclear, as the January edition was not sent and the client's goals are undefined.
  - **Context:** The newsletter is a low-lift task (\~0.5 days/month) for an audience of existing customers and trade show contacts.
  - **Decision:** Liz will include the newsletter in the SOW and use the contract discussion to get a clear direction from the client.
