### GrowthX Business Model

The foundational thesis: websites are your most valuable asset. Content is the atomic unit. AI multiplies importance because you're now serving multiple audiences (humans + AI agents) across a longer tail of queries.

**Delivery model:**
- 8-week strategy sprint to build context files
- Forward-deployed AI engineers, design engineers, strategists
- Context engineering = marketing
- Map all buyer questions across the funnel
- Calibrate quality definition (like data labeling)
- Editorial function that evolves into labeling function
- Force the last mile—GrowthX publishes, doesn't just hand off
- Ingest signals, prioritize, optimize, repeat

**Pricing:** $210K average contract value. Two kickoffs per week (intentionally selective).

### Technology and Platform

Built on Output AI—an agentic framework built on coding agents (not drag-and-drop). Marcel's CTO built If This Then That. They learned that 0.1% of users actually use workflow tools properly.

**Why code-based:**
- File system with runtime layer abstracted
- Full observability and evals
- "Anything we want to build is a command in Claude Code that turns into a file system with evals"

**Platform launch:** Next month. Moving from pure service to hybrid model—forward-deployed operated + client-managed command center.

### Strategic Priorities for 2026

2. **End-to-end platform**: Context → strategy → creation → optimization. Customer access next month, then public.
3. **Output AI open source**: NPM package. Developer adoption for workflows-as-code.
