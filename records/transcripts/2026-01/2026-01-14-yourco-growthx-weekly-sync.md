## Key Topics

### Content Strategy & Performance

  - **Problem:** Traffic is up (+70.4% MoM), but conversions remain low, indicating a need for higher-intent content.
  - **Solution:** Focus on three content directions:
    1.  **"Top X Tools" Listicles:** Create definitive lists for key use cases (e.g., "Top Communication Tools for Hospitality").
    2.  **Deep-Dive Q\&A:** Answer specific, high-intent questions within existing articles.
    3.  **Direct Competitor Comparisons:** Create content directly comparing Yourco vs. Udex.
  - **Execution:** Double down on top-performing subclusters (Scheduling, Call-Offs, Surveys) and under-represented topics (Compliance, SOPs).

### Yourco vs. Udex: Key Differentiators

  - **Security:** Yourco is SOC 2 compliant; Udex is not.
  - **Enterprise-Grade Admin:** Yourco offers a multi-level structure (Super Admin → Admin → Manager) for HQ oversight and local control.
  - **Communication Modes:** Yourco supports both one-way blasts and two-way conversations; Udex is primarily two-way.
  - **"Frontline Intelligence":** Yourco's core brand message, emphasizing data-driven insights beyond simple communication.

### New LLM Tracking Tool: `checkthat.ai`

  - **Migration:** Replacing Scrunch with GrowthX's internal tool, `checkthat.ai`.
  - **Key Advantages:**
      - **Historical Data:** Provides 90 days of data immediately upon setup.
      - **Cost-Effective:** No per-prompt fees.
      - **Direct Access:** Yourco will get admin-level access for self-service analysis.
  - **Status:** In beta; Yourco's public profile description is auto-generated and inaccurate, but can be updated by an admin.

### Link Exchange Opportunity

  - **Proposal:** A mutual link exchange between Yourco and `checkthat.ai` to build domain authority.
  - **Benefit for Yourco:** Gain high-quality backlinks from a relevant domain.
  - **Benefit for `checkthat.ai`:** Build its own domain authority.
  - **Client Connection:** Explore a link exchange with client Engine (travel/expense management) due to a similar frontline audience.
