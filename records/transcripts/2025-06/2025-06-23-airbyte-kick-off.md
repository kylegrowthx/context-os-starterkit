---

## Key Topics

### Airbyte's Market Position and Target Customers

  - Moving upmarket to target larger enterprises and traditional industries (healthcare, financial services)
  - Key value propositions:
      - Infrastructure modernization (e.g. replacing legacy ETL tools)
      - On-premise/hybrid deployments for data sovereignty requirements
      - Cost savings (e.g. C-Man saving $700k/year switching from Azure Data Factory)
      - Enabling AI/ML workflows by consolidating data sources

### Current Content Strategy and Performance

  - Created 10k+ pages in last 1.5 years
  - Some pages performing well, others need optimization
  - Need to refresh content every 60-90 days to keep up with AI platform changes
  - Successful content types:
      - Data glossary
      - Competitor comparison pages
      - Tutorials and use cases
      - "Top X tools" listicles

### AI Workflow Capabilities

  - Fact-checking and content verification
  - Metadata generation from GitHub repos
  - User session analysis for product insights
  - Image generation and asset creation
  - Potential for automating white paper creation and sales enablement content

### Marketing and Sales Funnel

  - 70% of enterprise deals start with product trials
  - Outbound: 70% phone, 20% LinkedIn, 10% email
  - Email effective as marketing touchpoint to increase phone call success
  - Need for more personalized content and lead magnets

### Technical Infrastructure

  - Using Superbase (Postgres database) for managing large volumes of content
  - Challenges with traditional CMS for high-volume content updates
