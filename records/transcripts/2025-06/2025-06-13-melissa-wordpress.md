## Key Topics

### WordPress Backend Navigation

  - Melissa demonstrates navigating WordPress backend for blog post creation
  - Highlights importance of using new template (still in progress) to avoid inherited mistakes
  - Shows how to break up sections, typically done when inserting images

### Adding Elements to Posts

  - Demonstrates adding quote blocks: Templates \> Quote Style \> Quote One
  - Explains process for inserting and positioning elements within the post
  - Covers code snippet insertion, including current limitations and workarounds
      - Using Carbon for visually interesting code displays
      - Temporary solution: desktop version (iframe from Carbon) and mobile version (Elementor block)

### Styling and Formatting

  - Emphasizes removing extra code and spans when pasting from Google Docs
  - Advises against bolding H3s, as H2s are used for subheadings
  - Demonstrates how to add and format bullet points within posts

### Cover Image Creation in Figma

  - Walkthrough of na-mii.co's Figma template for blog post images
  - Process: Select layout \> Copy to working area \> Use slugified title for SEO
  - Text highlighting technique for new style
  - Importance of title case (APA style) for text in images

### Brand Guidelines and Fonts

  - Shared brand guidelines document for accessing official fonts
  - Instructions for downloading and installing required fonts (Be Vietnam Pro)
  - Tips on logo usage and color adjustments based on background

### Image Handling in Figma

  - Demonstrated how to change background images in templates
  - Importance of desaturating images to maintain brand color consistency
  - Tips on cropping and positioning screenshots or other visuals
