## Key Topics

### 8-Week Strategy Sprint Framework

  - Developed baseline 8-week plan for client onboarding and execution
  - Key phases: kick-off, learn quickly, set foundation, develop pillars, calibrate, publish
  - Emphasis on being responsive, unblocking issues, and ingesting context upfront
  - Goal is to shape strategy quickly and move to execution to demonstrate value

### Calibration Process

  - Avoid going straight to final content without understanding client preferences
  - Use "tasting menu" approach with different content styles/tones for client feedback
  - Develop rubric based on client preferences to guide future content creation
  - Tailor calibration to specific content types (e.g. listicles vs how-to articles)

### Client-Specific Updates

  - GetPod: Need to redo calibration and demonstrate clear value in next 2 weeks
  - Engine: Updating Airtable with assignments, tagging content clusters
  - Hypergrowth: Custom approach, waiting on resource allocation
  - Biologica: Handling confidentiality concerns with shared documents

### Artifact Creation

  - Developing reusable, AI-ready artifacts for each client project
  - Includes company context, research summaries, brand guidelines, etc.
  - Can be exported as PDFs and used in AI tools like Claude or ChatGPT
  - Provides value to clients even if engagement ends early
