## Key Topics

### Gitpod Rebranding to ONA

  - Driven by evolving product vision beyond cloud development environments
  - Creates space for larger product ambitions and re-energizes stakeholders
  - Leverages Gitpod's security architecture to offer secure sandbox for software agents
  - Targets large enterprises in regulated industries as ideal customer profile (ICP)

### Product Differentiation

  - Self-hosted in customer's VPC but vendor-managed for reduced operational overhead
  - Provides isolated, secure environments for running AI agents at scale
  - Integrates with existing tools and workflows, minimizing disruption
  - Offers both centralized control and individual developer flexibility

### Competitive Landscape

  - Main CDE competitor is Coder, differentiated by Gitpod's managed service model
  - New competition from AI coding assistant companies (e.g. GitHub Copilot, Cursor)
  - Positioning as foundational layer for securely running any AI coding tools

### Target Audience and Messaging

  - Primary ICP: 1000+ employee companies in regulated industries with engineering productivity mandates
  - Secondary: Individual developers as potential champions within enterprises
  - Challenge: Appealing to both enterprise buyers and individual developers
  - Considering shift from "CDE with agents" to "AI development platform for humans and agents"

### Go-to-Market Strategy

  - Leveraging strategy sprint to reset approach given recent product changes
  - Exploring content strategies to educate market on agent use cases and security considerations
  - Investigating programmatic content generation for long-tail keywords related to development environments
  - Considering affiliate site strategies to combat reduced organic traffic from AI-generated search results

### Product Workflow

  - Working to simplify self-serve onboarding with "Gitpod hosted compute" launch
  - Ideal flow: One-click setup from GitHub, automated environment configuration by agent
  - Emphasizing parallelism and ability to run multiple agents simultaneously
  - Integrating with popular IDEs and tools while offering enhanced security and scalability
