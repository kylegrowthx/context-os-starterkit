## Key Topics

### Company Projects Overview

  - Atlas (atlas.growthx.ai): Rails app for client management
      - Features: Artifacts (canonical documents), Projects (pipelines and workflows)
      - Pipelines: YAML-based, automated execution with human review steps
  - Flow: TypeScript app for workflow execution
      - Uses Temporal for workflow orchestration
      - Workflows are code-based (TypeScript) with XML documentation
  - Flow Studio: Rails app for workflow creation and visualization
      - No database, consumes data from Temporal

### Workflow and Pipeline Structure

  - Workflows are coded in TypeScript with activities, prompts, and XML documentation
  - Pipelines are defined in YAML, specifying steps that map to workflow functions
  - Steps can send data to the output, allowing flexible data composition
  - Two main pipeline types: editorial creation and publishing

### Future Development Plans

  - Improve Atlas for agency scalability (target: $10-20M revenue)
  - Develop a task hub for efficient task allocation to writers and publishers
  - Enhance the Flow product to become a general-purpose AI feature development platform
  - Create a more user-friendly workflow creation environment (similar to Bolt or Webflow for React)
