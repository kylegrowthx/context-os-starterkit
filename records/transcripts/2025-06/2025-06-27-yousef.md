---

## Key Topics

### Core Application Access Removal

  - Gmail: Suspend user in Google Workspace admin
  - Slack: Suspend user in Slack admin
  - Notion: Remove from workspace, suspend in Organization Console
  - Airtable: Deactivate user in admin panel
  - Okta: Deactivate user in admin dashboard
  - 1Password: Suspend access in admin dashboard
  - Zoom: Deactivate user, option to transfer meetings
  - Loom: Deactivate user in Workspace settings
  - Fathom: Deactivate as "ex-employee" to retain shared recordings

### Google Workspace Detailed Cleanup

  - Security tab: Reset sign-in cookies, remove app-specific passwords
  - Remove iOS/Android/Chrome devices
  - Review connected applications for additional access points
  - Remove user from all groups

### Additional Considerations

  - Linear: Check if user was set up, remove if necessary
  - Deal: Only address if user was an admin
  - Air Ops: No longer in use, can be skipped
  - Atlas: No action required
  - Otter: Not actively used, can be ignored
