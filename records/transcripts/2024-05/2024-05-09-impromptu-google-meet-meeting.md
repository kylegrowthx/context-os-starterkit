---

## Key Topics

### Content Checklist

  - #### SEO Guidelines
    
      - Target 1 primary keyword + 3-6 secondary keywords
      - 4-6% keyword density
      - Include target keyword in first sentence
      - H1 title: 55-70 chars, start with keyword if possible
      - Use CoSchedule Headline Analyzer tool
      - 5-10 internal links, relevant anchor text
      - 2-5 external authority links
      - Optimize images (file names, alt text, original vs stock)
      - Compelling meta description with call-to-action

  - #### Writing Guidelines
    
      - Max 3-4 line paragraphs for scannability
      - Use Hemingway app to target \~7th grade reading level
      - H2/H3 header guidelines

### Brand Voice

  - Personable, straightforward, tenacious tone that adapts to situation
  - Follow brand style guide (e.g. sentence case, Oxford commas)
  - Aim for informative yet conversational voice, avoid overly simplistic or embellished

### Content Strategy/Taxonomy

  - Categorize by funnel stage (top/middle/bottom/sales enablement/user engagement)
  - Map to user needs/pain points
  - Top funnel = awareness (e.g. "what is payroll?")
  - Middle funnel = evaluation (e.g. "payroll tools comparison")
  - Bottom funnel = product-specific (e.g. "is HomeBase payroll right for me?")
