## Key Topics

### Comparison Article Generation Process

  - Marisol uses agentic article generation with focus on detailed assignment briefs
  - Custom prompt created for comparison/listicle articles, including:
      - Product features
      - Voice and tone
      - Content structure (main sections, conclusions)
      - Content strategy (TOFU, MOFU, BOFU)
      - SEO and keyword usage
      - Quality checklist
  - Process involves reverse-engineering competitor articles for optimal structure
  - Uses custom GPT (Context Engineering Applied) to optimize prompts under 600 tokens
  - Generates \~30 articles per week, prioritizing speed and efficiency

### Assignment Creation and Optimization

  - Uses Atlas for assignment generation with custom prompts
  - Adapts assignment direction based on article type (listicle, comparison, how-to)
  - Optimizes assignment briefs by removing unnecessary sections (e.g., hypothetical use cases)
  - Tailors content for senior developer audience, avoiding basic explanations

### Post-Editing and Refinement

  - Applies post-editing prompts for analysis and improvement suggestions
  - Uses Claude AI for technical validation and alignment with writing guidelines
  - Implements a "polishing prompt" for SEO optimization and readability
  - Applies Paul Graham-inspired style for more human-like, concise paragraphs

### Content Cannibalization Prevention

  - Weekly download of published articles to compare with upcoming content
  - Uses LLM to identify potential duplicates or articles that should be combined
  - Generates diverse comparison topics using AI-suggested competitor combinations

### Technical Accuracy and Product Updates

  - Custom prompt for technical validation of Augment Code features
  - Automated weekly changelog updates to maintain current product information
  - Aims to integrate changelog data into Atlas pipeline for up-to-date product features
