## Overview

- Vizcom accelerates zero→one and refinement: sketch→color block→shadows/highlights→photo-real; reversible back to sketch for easier edits
- Core toolkit: Workbench (Miro-like) + Studio (Photoshop-like), Instant Render, Modify (NL prompting), variation tools (form/color), animation, 3D→USDZ AR, full history
- Styles (formerly Palettes) = private, workspace-scoped custom models for brand/sketch/CMF consistency; “Bone” and Scott Robertson examples
- Verticals beyond product design: packaging, footwear/apparel, architecture/interior, concept art, gaming environments/characters

---

## Key Topics

### End-to-end workflow (double diamond fit)

  - Zero→one ideation: render from sketch, explore form/color, change expression via controls
  - Refinement: alternate views, direct on-image annotations → targeted edits
  - Communication: generate 3D, export USDZ, view in AR; create animations and in-context shots

### Workbench vs. Studio

  - Workbench: board of artifacts (Miro-like) → multi-asset exploration; same toolset available
  - Studio: focused canvas (Photoshop-like) → layers, sketching, compositing; persistent history per asset

### Instant Render (fidelity ladder + history)

  - Progressive passes: sketch → color block → highlights/shadows → realistic
  - Select results into layer stack; full history retained → designers can compare/branch
  - Reverse fidelity: convert realistic/3D back to “wide sketch” for shape-first iteration

### Image compositing + adjustments

  - Magic Erase → fix blemishes; Remove Background → isolate concepts
  - Enhance + photographic controls → exposure/contrast, Lightroom-like tweaks
  - Single environment replaces fragmented toolchain (image editing + AI + 3D)

### Form and color variation

  - Form “dial” axes: organic↔geometric, simple↔complex; preset terms (e.g., heavy/dense) and custom descriptors (e.g., hiking shoe → beach sandal)
  - Output gallery shows spectrum from sketch-faithful to inventive
  - Color variation: auto-extract palette; shuffle or map to preset/custom palettes; assets saved for reuse

### Modify (natural language prompting)

  - Contextual edits via plain text: e.g., “place sneaker on foot,” “standing on a tennis court,” “blur background”
  - Rapid in-context visualization → supports review/sell-through

### Animation

  - Prompted motion: e.g., walking on court; camera behaviors (tracking/bounce)
  - Turntable generator with start frame control → angle selection for continued 2D edits
  - Any frame can be captured back to canvas for higher-fidelity still refinement

### 3D generation + AR

  - Generate 3D from concept; adjustable fidelity; compositing-friendly camera control
  - Export USDZ → iOS AR viewing; value for scale-sensitive categories (furniture, footwear)
  - Example: sole unspecified yet plausible pattern generated → jump-off for resketching

### Annotation-driven edits

  - Draw callouts/lines on image → prompt-targeted changes (e.g., “raise sock to red line”)
  - Scrubbable generations show how edits track across outputs

### Styles (formerly Palettes)

  - Private to workspace; act as custom mini-models
  - Use cases: brand language (e.g., Dyson-like forms), sketch style (Scott Robertson’s vehicles; “5” detail learned), CMF modes (e.g., “Bone” flat gray shaded volumetrics)
  - Strength slider: 100% → sketch-faithful; lower → more prompt-led creativity (temperature analog)

### Users and positioning

  - Core: product designers (industrial, footwear/apparel)
  - Adjacent: packaging/labels, architecture/interior, concept art; gaming teams → scenes/environments/characters
  - Messaging: tool supports day-to-day tasks (CMF studies, angle selection, context comps), not “design a whole car” claims
