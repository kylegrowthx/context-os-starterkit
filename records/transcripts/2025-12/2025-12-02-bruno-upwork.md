## Key Topics

### Project Goal & Scope

  - **Objective:** Experiment with website templates for various niches to gauge user interest.
  - **Output:** 4 visually distinct variations of a single base template each week.
  - **Tool:** Lovable (Bruno can use Cloud Code or in-app chat).
  - **Quality Standard:** Production-ready for immediate use.
      - **Functional:** All links must work.
      - **Visual:** No duplicate images or mobile layout issues.
      - **Cohesive:** Consistent styles (typography, colors) across all pages.
      - **Note:** Internal style guides must be removed from final deliverables.

### Test Workflow

  - **Rationale:** A test week will validate the weekly 4-variation commitment, as the scope is larger than Bruno's previous 2-project/day pace.
  - **Schedule:**
      - **Wednesday:** Georgemaine provides the base template.
      - **Monday:** Bruno delivers 4 variations.
  - **Communication:** Email for the test week. A formal Notion/Slack workflow will be implemented if the test is successful.
