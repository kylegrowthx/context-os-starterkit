## Overview

- AdRoom offers AI-powered image generation, editing, and resizing tools suitable for creating blog featured images
- The platform integrates brand assets (logos, colors, fonts) and guidelines to maintain brand consistency
- Custom prompts with AI enhancement allow for diverse image styles, from realistic to illustrated graphics
- Upcoming platform updates will improve UI/UX and expand functionality (e.g., Figma integration)

---

## Key Topics

### AdRoom Platform Overview

  - Left panel navigation: folders, projects, assets, brand styles
  - Brand setup: upload logo, colors, fonts, website, guidelines, description
  - Four tailored workflows: ad copies, images, videos, landing pages
  - Custom prompt model recommended for advanced, easier-to-use image generation

### Image Generation Capabilities

  - Stock image creation from text prompts
  - Product creative generation by placing product on blank canvas and adding background/scene
  - AI understands product context, maintains product integrity while generating complementary elements
  - Supports various image styles: realistic, illustrated, digital graphics
  - One-click resizing for multiple social media formats

### Image Editing Features

  - Integrated AI editor with Photoshop file support
  - Magic options: eraser, segmentation for object/color manipulation
  - Text generation and styling with brand font/color suggestions
  - Layering system similar to Photoshop

### Upcoming Platform Enhancements

  - Improved UI with thumbnail previews and expanded generation options
  - New word editor with streamlined left-panel tools
  - Bulk mode for efficient multi-SKU design creation
  - Figma integration planned for next 1-2 quarters
