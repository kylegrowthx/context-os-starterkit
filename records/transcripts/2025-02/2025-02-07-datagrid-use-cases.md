## Summary

GrowthX leadership met with Datagrid (represented by Thiago da Costa) to identify and plan use cases for Datagrid's agentic AI platform beyond its core construction industry focus. The team aligned on a programmatic SEO and image classification use case with Adventures Group (travel/hospitality), planned for a March 20th webinar showcase, and confirmed next steps including webinar promotion via LinkedIn paid ads and confirmation of image storage location with the Adventures Group team.

## Context

Datagrid is an agentic AI platform that processes structured and unstructured data to automate knowledge work. The company has deep traction in construction (site safety, material tracking) but seeks to expand into adjacent verticals. Thiago da Costa, who leads product, presented several use cases: conference planning optimization, image classification and enrichment, transcript analysis, and Upwork productivity tracking. GrowthX sees an opportunity to develop a reference case with Adventures Group (a travel/hospitality client with large image repositories) that demonstrates Datagrid's capabilities while showcasing GrowthX's SEO integration expertise. The webinar aims to position both Datagrid's agentic platform and GrowthX's differentiated API integration against competitors like AirOps.

## Key Topics

### Datagrid's Agentic AI Capabilities

- Demonstrated ability to process and analyze large datasets (e.g., conference planning, customer visits)
- Can perform complex tasks like image classification, content analysis, and data enrichment
- API allows for integration into existing workflows and systems
- Platform costs scale with outcomes, not licenses — agents reduce knowledge work cost to near-zero

### Use Case Development

- Focus on programmatic SEO and image classification for marketing applications
- Adventures Group identified as potential showcase client (travel/hospitality industry)
- Use case to involve classifying and enriching image database, then using AI to select appropriate images for content
- Requirements: simple to set up, deliverable value quickly, no custom integrations required, usable via chat interface

### Datagrid's Cross-Industry Capabilities

- Conference and travel planning: agent organizes customer visits and conference dates across multiple spreadsheets (executed in seconds vs. hours for manual EA work)
- Image classification at scale: 6 million construction site photos processed to identify work activities, materials, and safety equipment
- Transcript analysis: ingests Google Drive files to extract topics and enrich data
- Upwork productivity tracking: analyzes freelancer screenshots to identify time allocation

### Webinar Planning

- Scheduled for March 20th (Thursday morning)
- Will feature both Datagrid's platform capabilities and GrowthX's API integration
- Aim to attract marketers, content creators, and SEO professionals
- Jimmy Pal to lead webinar organization; LinkedIn paid ads targeting marketers and SEO pros planned

### Competitive Positioning

- Positioning against competitors like AirOps, highlighting Datagrid's superior agentic AI capabilities
- Emphasis on cross-industry applications and API integration flexibility
- Demonstrates how GrowthX's CheckThat integration can be plugged into Datagrid workflows

### Adventures Group Image Classification Use Case

- Large image repository (travel/hospitality content) stored in Google Drive
- Workflow: pull images from Drive, classify by context, enrich metadata, AI selects appropriate images for blog posts and content
- Agents can extract images based on activity, location, and content type
- Replicable pattern applicable to any content-heavy industry
