## Key Topics

### Current Client Use Case

  - 15 articles/week, translated to Spanish
  - 14-18 images per article, mostly from Shutterstock
  - Three different websites (locations) involved
  - Image resizing and WebP conversion required

### Proposed AI Agent Functionality

  - Extract relevant photos based on specific article content
  - Potential for automated image placement (e.g., every X lines)
  - Workflow: AI agent column processes dataset of photos, outputs text with embedded image URLs
  - Ability to batch process multiple articles quickly (e.g., 1000 articles in minutes)

### Webinar Preparation

  - Use publicly available dataset of royalty-free images
  - Create sample articles to match the image set
  - Consider including SaaS-specific use cases to align with Tiago's interests
  - Possibility of showcasing multiple examples (e.g., general and SaaS-specific)

### Potential Additional Use Cases

  - Finding specific elements in images/charts/graphs (e.g., error messages in screenshots)
  - Analyzing user recordings for UX/UI insights
  - Agency-focused solutions: client reporting, data integration
  - Analytics synthesis across multiple tools and data sources
  - Grading deliverables from contractors (e.g., assessing image quality and placement)

### Webinar Theme and Structure

  - Overarching theme: "How to use agents with images"
  - Combine embedding images in articles with other image-related AI agent use cases
  - Goal: Inspire audience with varied use cases and possibilities
