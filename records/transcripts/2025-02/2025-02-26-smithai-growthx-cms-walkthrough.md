## Key Topics

### Webflow CMS Access and Navigation

  - Access editor mode: Add "?edit" to Smith.ai URL
  - Collections contain all editable content (blog posts, categories, tags)
  - Preview changes using arrow button; custom code elements don't show in preview
  - Use incognito window to view live site while editing to avoid sync errors

### Blog Post Creation and Editing

  - Fields include title, slug, SEO metadata, post summary, body content
  - Use proper formatting in Google Docs/Word for easy pasting into Webflow
  - Add custom elements (photos, videos) via the "+" button
  - Set featured status, categories, tags, author (currently Maddy), and publication dates
  - SEO last update field purpose to be clarified

### Image Handling

  - GrowthX responsible for adding and positioning images in blog posts
  - Use TinyPNG for image optimization; convert to WebP format
  - Set blog images to full width using the second option in image settings

### Publishing Process

  - Use "Save Draft" for internal review
  - "Publish Now" or "Schedule" for immediate or future publishing
  - Always use the green "Publish" button for individual pages, never the blue site-wide button

### CTA and Email Handling

  - Automatic CTA appears on all blog posts; chat-specific CTAs available
  - For manual email CTAs, use the "email" link type in Webflow editor
