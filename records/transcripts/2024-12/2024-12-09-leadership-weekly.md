### Project Management Tool Decision

  - Team debating between continuing with Notion vs switching to Basecamp
  - Key issues with Notion: weak task management, lack of proactive notifications
  - Basecamp pros: more opinionated structure, better for long-lived projects
  - Action: Marcel and Jimmy to write opposing viewpoints to evaluate options
  - Considering hiring consultant to implement chosen solution and train team

### Engineering Updates

  - Posting 2 new positions: full-stack dev for AirTable/AirOps replacement, backend dev for API work
  - Marcus joining as senior full-stack engineer to work on AI/prompt engineering

  - Working on financial projections and capacity planning models

### AirOps Strategy

  - Developing approaches for accounts wanting workflows in their AirOps instance (e.g. Tyro, Hagen)
  - Options include:
    1.  Using API calls to trigger our workflows without exposing full IP
    2.  Providing simplified versions of workflows directly in client accounts
    3.  Balancing visible usage in client accounts while protecting IP
  - Preparing contingencies in case AirOps relationship deteriorates further

  - Marcel to continue running hiring process with Molly's help; Kyle to be involved in final stages
