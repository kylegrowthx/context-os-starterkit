## Key Topics

### CMS Navigation and Basic Functions

  - Access CMS via top menu: Insert \> Layout \> Text \> CMS
  - 'Block' section is primary area for content creation
  - Creating new items auto-generates slugs; can be manually overridden
  - Date selection, category assignment, and image upload with focal point feature
  - Simple content editor with formatting options (headings, lists, images, quotes, videos)
  - Draft/Published status visible; one-click publishing

### Content Structure and Formatting

  - Title, intro text, and main content areas available
  - H2 recommended for main title, H3 for subheadings
  - Easy insertion of images, bullet lists, and other media
  - Alt text can be added to images for accessibility

### Image Handling

  - Direct upload without separate database management
  - Framer auto-resizes images for mobile/desktop
  - Recommendation: keep images under 150KB for optimal performance
  - Discussion planned for image sourcing (custom vs. generated)

### Category Management

  - Current limitations on adding new categories
  - Plan to align categories with topic clusters
  - Backend work required for category updates and overview page

### SEO and Page Settings

  - Custom slug creation possible for keyword optimization
  - Page settings allow for preview customization, code insertion, and description editing
  - Changes in page settings apply globally to all blog pages
