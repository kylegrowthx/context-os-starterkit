## Key Discussion Points

**Candidate Evaluation Framework**
- Three-question approach focusing on: 1) Ability to clarify briefs and translate to actionable strategy, 2) AI tool proficiency and prompt iteration, 3) Systems thinking and process discipline
- Red flags: passive approach to brief clarification, generic AI usage without understanding, reliance on instinct vs. systems
- Value: candidates with initiative, curiosity about technology, self-starting ability, and bias to action

**Best Hire Example**
- Opera Software hire: showed AI enthusiasm despite limited hands-on experience
- Progressed from content manager to senior leader managing others
- Identified process issues (Trello card visibility causing missed edits)
- Implemented color-coded system to improve workflow clarity and reduce errors
- Demonstrated initiative, problem-solving, and collaborative improvement mindset

**Five-Day Onboarding Plan**
- **Day 1**: Foundational - editorial standards, workflows, guidelines, voice, brief creation
- **Day 2**: Production workflows - templates, demos, AI examples, peer review
- **Day 3**: AI prompt skills - frameworks, troubleshooting, prompt iteration, library overview
- **Day 4**: Client examples - pattern analysis, rewrites, tone optimization, feedback
- **Day 5**: Live client work - brief refinement, production support, readiness

**Key Onboarding Milestones**
- AI prompt skills proficiency - critical for scaling content
- Brief creation clarity - functional, succinct, clear communication of requirements
- Difficulty in either area signals fit issues or process revision needs

**Performance Tracking Recommendations**
- Build internal dashboards using formulas and time metrics
- Track AI usage (target 70-80%), draft timing, revision rates
- Monitor impressions, clicks, session data for quality indicators
- Two-week trial periods for improvement interventions

**Intervention Strategies**
- **Overuse of AI**: Set usage limits, explain balance importance, quality implications
- **Underuse of AI**: Review workflows, provide prompt templates, set usage targets above 70%
- **High revision rates**: Diagnose via conversation to determine if cause is resource overload, tool understanding, or brief clarity

**Strategic Context - Platform Adoption Challenges**
- Many editors struggle because they haven't learned to use the proprietary AI platform properly
- Cultural shift needed from editorial mindset to AI-leveraged content creation
- Need to balance universal processes with client-specific adaptations
- Issue often framed as "using the pickaxe from the wrong end"
- Platform is nascent and requires strong onboarding and ongoing training

**Success Indicators (90-day assessment)**
- Editor feedback on workflows and processes
- Output improvements - time to publish reductions (e.g., 5 days to 3 days)
- Production quality assessments
- Communication of challenges and bottlenecks
- Workflow effectiveness measures

**AI Prompt Engineering Test**
- Proposed addition to hiring process for specific, practical AI skill assessment
- Should go beyond generic AI questions to test prompt iteration and problem-solving
- Evaluates comfort with prompt frameworks and troubleshooting
