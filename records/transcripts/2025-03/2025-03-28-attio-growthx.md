## Overview

- GrowthX plans to implement Default for lead qualification and routing, with a webhook to Attio
- Attio's workspace object can be utilized for comprehensive customer success management
- Attio offers flexibility in creating custom views (e.g., lists, Kanban) for project/customer tracking
- Attio is open to providing discounted rates for additional "lite" user seats in the future

---

## Key Topics

### Lead Qualification and Routing Solution

  - GrowthX currently bottlenecked by unqualified leads reaching the founder
  - Exploring options: Chili Piper, RevenueHero, Default
  - Default emerged as preferred choice due to features and pricing
  - Attio integration possible via webhook or Zapier, negating need for native integration

### Attio for Customer Success Management

  - Workspace object represents customer accounts, storing key information
  - User object distinct from contacts, represents product users
  - Lists used for various purposes: onboarding, renewals, account health monitoring
  - Sequences tool available for email campaigns, integrated with workflows
  - Activity feed shows product usage updates, list changes

### Attio Customization and Flexibility

  - Custom objects can be created (e.g., for projects)
  - Lists can be used to create Kanban views for project/customer tracking
  - Tasks can be associated with workspaces or companies, depending on team process

### User Management in Attio

  - Currently no "lite" user option, but Attio open to providing discounted rates
  - GrowthX considering adding \~10 additional users in the future
