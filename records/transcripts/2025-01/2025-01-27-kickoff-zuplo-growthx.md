## Key Topics

### Zuplo's Product and Market Position

  - API management tool focused on simplifying API development
  - Core product is an API gateway, expanding into broader API management features
  - Main competitors are other API gateway and management providers
  - Target industries include FinTech, Web3/crypto, and AI companies

### Current Content Strategy

  - Mix of AI-generated content, guest posts, and in-house created articles
  - Using tools like SEMrush for keyword research and ChatGPT for content generation
  - Manual process for internal linking and content optimization
  - Occasional video content creation with developer relations team

### Traffic and Conversion Metrics

  - Seeing linear correlation between SEO traffic and demo bookings
  - Roughly 1 demo booking per 500 visitors per week
  - Challenge in de-anonymizing users due to Gmail signups

### Target Audience and ICPs

  - Enterprise: CTOs, VPs of Engineering, Directors (500+ employee companies)
  - Startups/Scale-ups: Founders, CTOs
  - Self-serve: Software engineers, backend developers, API developers

### Content Goals and Priorities

  - Increase organic traffic to compete with established competitors
  - Focus on programmatic SEO for API directories to drive top-of-funnel traffic
  - Improve backlink profile and domain authority
  - Create content that appeals to both enterprise decision-makers and individual developers

### Brand Voice and Style

  - Aiming for a "cool", maverick-like tone with strong opinions
  - Incorporate reverent humor and witty elements
  - Provide plenty of examples and explanations, like "your cool uncle explaining APIs"
