## Key Topics

### Current Image Implementation Status

  - Fatima has uploaded 3 images so far
  - \~23 pages need images (21 in draft, 3 published)
  - Client specified GIFs for use cases at the bottom of pages
  - No cover images currently at the top of pages

### Programmatic vs. Manual Image Selection

  - Marcel explained that programmatic image generation is not ideal for this project:
      - Automated search and placement possible, but not actual image creation
      - Good AI-generated images (e.g., Midjourney) lack API access
      - For 23 pages, manual selection is faster than building a workflow
      - Programmatic approach more suitable for 3000+ pages

### Stock Image Resources

  - Fatima has access to Unsplash and Pixels for royalty-free images
  - Getty Images suggested as a potential source
  - Fatima confident in finding suitable images from these resources

### Page Design Considerations

  - Current design looks "bland" without cover images
  - Suggestion to propose adding cover images to improve aesthetics
  - Client has decided on GIF placement, but top of page remains empty
