## Key Topics

### Creating New Blog Posts

  - Go to Template Blog Post content type to create new entry
  - Add title, SEO fields, featured image
  - Select category, author, publish date
  - Add excerpt (100-150 chars) and body copy in rich text field
  - Can embed images, links, conversion panels

### Images and Visuals

  - Currently using illustrations, looking to incorporate more photography
  - For header images, stock photos or screenshots can work depending on topic
  - GrowthX can propose images, Webstacks can also provide existing assets
  - Exploring AI image generation tools like Midjourney

### Content Workflow

  - GrowthX generating \~10 articles/week, can share topics/outlines ahead of time
  - Initial batch of outlines to be reviewed by Webstacks for alignment
  - Ongoing cadence (weekly/monthly) for topic sharing TBD
  - Webstacks OK with GrowthX publishing directly, then making revisions
