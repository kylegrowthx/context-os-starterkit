## Overview

- You.com is pivoting to focus on AI search infrastructure and API-first approach for developers
- Current challenges: low brand awareness, limited presence in developer conversations
- Key differentiators: vertical domain indexes, composable API, fresh news index
- Need to develop benchmarks, use cases, and content strategy to compete with established players

---

## Key Topics

### You.com's Strategic Shift

  - Transitioning from consumer brand to enterprise/developer focus
  - Redefining category as "AI search infrastructure" with two key components:
    1.  API access (SDK, MCP)
    2.  Vertical domain indexes (narrow, deep information)
  - Primary target: Developer community (VP Engineering, Product)
  - Secondary: CIO community (less technical, need end-to-end solutions)

### Competitive Landscape

  - Main competitors: Anthropic, Perplexity, traditional SERP API providers
  - Differentiators:
      - Composable API (flexibility for developers)
      - Fresh news index
      - Vertical domain indexes (e.g., hospitality, legal, retail)
  - Challenges:
      - Commoditization of basic search API
      - Need for clear differentiation in accuracy, latency, freshness, or cost

### Customer Acquisition Strategy

  - Trigger events:
      - Bing shutting down their service
      - Competitors reducing API call limits
      - Need for breaking news capability
  - Evaluation process critical:
      - Human preference benchmarks
      - Custom evaluation sets
      - Open vs. closed evaluations

### Content Strategy

  - Developing use case-specific content (e.g., retail deep dive)
  - Creating guides on when/where to use You.com APIs
  - Focusing on SEO/AEO for basic terms to build authority
  - Leveraging multiple content pieces to dominate AI-generated answers
  - Exploring paid newsletters and gated content for lead generation

### Collaboration with GrowthX

  - 8-week strategy sprint to align direction, strategy, and quality definition
  - Deep dives planned with various team members
  - GrowthX to provide "What We Heard" document and follow-up actions
