## Key Topics

### Airtable Database Structure and Enhancements

  - Marcel duplicated Kerry's existing database and demonstrated advanced features:
      - Created relational tables for authors, categories, and other metadata
      - Implemented rollups to aggregate data (e.g., post counts by category)
      - Added fields for organic traffic, keywords, and cost data
  - Kerry expressed excitement about the potential for automatic gap analysis and deeper performance insights

### Data Import and Integration Challenges

  - Identified need for WordPress XML export to retrieve original publication dates
  - Discussed limitations with Airtable's import capabilities (5MB file size limit)
  - Explored possibilities for integrating Google Search Console data
  - Acknowledged manual data entry challenges for large datasets (943 URLs)

### Future Automation and CMS Migration

  - Plans to automate processes once Homebase migrates to Webflow CMS
  - Discussed potential for direct integration using Zapier, though noted high costs for frequent updates
  - Aim to extract content body directly from CMS rather than using current scraping and LLM processing method

### Content Strategy and Analysis

  - Kerry explained their content categorization system:
      - Product funnel: Aligns content with specific product areas (e.g., payroll, scheduling)
      - Funnel stage: Indicates where content fits in the buyer's journey (top, middle, bottom)
  - Discussed potential for analyzing content performance by various dimensions (product, funnel stage, vertical, persona)

### fwd.digital Business Update

  - Marcel shared that fwd.digital now has 14 customers and is scaling rapidly
  - Considering raising capital to accelerate growth and enhance customer service
  - Expanding team with graphics professionals and account managers
