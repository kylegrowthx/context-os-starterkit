  - **Production Status:**

      - In Review: 8 articles.
      - Delivery: 5 articles this week.
  - **Performance Metrics:**
      - Reporting is now monthly for better trend analysis.
      - Overall traffic is trending up.
      - The Model Hub is volatile due to a low page count.
      - **Key Insight:** Existing blog pages are already ranking \#1 for "model vs." keywords (e.g., DeepSeek vs. O1), validating the Model Hub's future traffic potential.
      - Competitor Arise is losing keywords at the same rate as Galileo.
      - LLM referral traffic increased, with GPT remaining the top referrer.
  - **Strategy Shift:**
      - **Priority:** Shift focus from competitor articles to new content for the Model Hub and the "Eval Engineering" cluster.
      - **Rationale:** This allows the team to fully digest new product marketing assets before redrafting competitor content.

### New Site Structure: Hub-and-Spoke Model

  - **Problem:** The current flat site structure is difficult to navigate and dilutes SEO value across many individual pages.
  - **Solution:** Adopt a hub-and-spoke model to create cohesive user journeys and consolidate SEO authority.
  - **Implementation Plan:**
      - **Pillar Pages:** Create a central index page for all pillars.
          - **Rationale:** Provides a cohesive learning pathway, setting up the December course launch.
          - **New Structure:** Consolidate subtopics (e.g., "AI Agent Evaluation") under a single parent pillar page ("AI Evaluation") to prevent navigation sprawl.
      - **Model Hub:** Create a central index page for all model comparisons.
          - **Rationale:** Consolidates traffic and SEO value as the hub grows.
      - **Competitor Hub:** Create a central index page for all competitor comparisons.
          - **Rationale:** Essential by Q1 2026, when the number of pages is projected to reach \~20.
          - **Model:** Use the Gusto competitor page as a template, adapting its structure (e.g., summary tables) while avoiding its poor UX (e.g., overly long case studies).
  - **Technical Note:** Jackson will consult the web dev agency on Framer's limitations for implementing this new structure.

  - Conor Bronsdon announced his departure from Galileo, with his last day being next week.
  - He will take a break before starting a new role in December.
