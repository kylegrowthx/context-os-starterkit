      - **Time on Site:** 1m 23s
      - **Key Events:** Steadily increasing, especially "shop directory views."
  - **December Traffic:** Historically stable, with no expected seasonal dip.

### New Project: LLM-Driven FAQ Strategy

  - **Goal:** Capture Answer Engine Optimization (AEO) traffic by making content easily consumable for LLMs.
  - **Hypothesis:** LLMs are "lazy" (compute-constrained), so providing simple, structured Q\&A content will improve visibility.
  - **Model:** Based on a successful experiment for Engine.com, which used Reddit scrapes to identify primary and related questions.
  - **Implementation:**
      - Create a new "FAQs" category in the blog.
      - Use Yoast's built-in FAQ schema to provide structured data for AEO.
      - **Rationale:** The blog is crawled more frequently for recency, which is a strong AEO signal.
  - **Content Sourcing:**
      - **Initial:** Non-branded questions identified via Reddit scrapes.
      - **Future:** Insights from sales/SDR calls to inform B2B-focused questions.
  - **PR Angle:** Position the project as an environmentally friendly initiative, as optimizing for LLMs reduces compute power.

### Project Logistics

  - **Case Studies:** The content pipeline is empty. All 71 case studies from the shared spreadsheet are complete.
  - **Scrunch Access:** Jason has direct access to the Scrunch account for self-service reporting.
      - **Purpose:** Measure the FAQ project's impact on visibility by comparing metrics against a baseline.
