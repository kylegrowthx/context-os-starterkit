## Key Topics

### Programmatic SEO Page Implementation

  - Pages will be built as singletons using Next.js app router
  - Existing components and Tailwind CSS can be reused from current site
  - No need for Keystatic CMS integration initially; content can be hardcoded
  - Will showcase real-time conversations with AI agents, not pre-recorded demos
  - Different scenarios (transfer call, end call, recommend service) can be implemented using separate assistant configurations

### Development Process

  - Clone repo, run locally, integrate Kirkland's existing demo
  - Submit pull requests for review
  - Look at existing singleton pages (e.g., startups, build challenge) for reference
  - Automatically regenerating sitemap when adding new singletons

### Vapi SDK Integration

  - Can change assistant config by sending JSON
  - Showcase developer-friendly features that highlight SDK capabilities
  - Implement live transcript and mid-call control functionalities

### Content Publishing Workflow

  - Fixed GitHub and Vercel integration issue
  - Content team can now publish directly through Keystatic without touching GitHub/Vercel
  - Simple category/tag additions possible through config file edits
