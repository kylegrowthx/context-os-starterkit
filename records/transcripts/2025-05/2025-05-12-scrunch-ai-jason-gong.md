## Key Topics

### Scrunch AI Platform Overview

  - Monitors brand mentions across AI platforms (ChatGPT, Google AI, Perplexity)
  - Key metrics: Presence (mention frequency), Position (placement in response), Sentiment
  - Prompts run every 3 days against selected AI models
  - Features include AI traffic analysis, source tracking, and AI-generated insights

### Data Visualization and Analysis

  - AI traffic tab: Looker Studio dashboard for Google Analytics integration
  - Sources tab: Ranks top-referenced sources by AI, with impact scoring
  - Insights tab: Flags opportunities and recommendations for improvement
  - Site audit feature: Shows how AI perceives webpage content

### Prompt Management and Generation

  - Users can manually add, bulk upload, or AI-generate prompts
  - Prompts can be tied to specific personas or key topics
  - Best practices depend on business breadth and focus areas

### Pricing and Package Details

  - Starting at $300/month for 3 orgs, $500/month for 5 orgs
  - 750 prompts shared across chosen platforms
  - Bulk discounts available for more brands/prompts
  - Upcoming: Industry-wide shared prompts at no extra cost

### API and Integration

  - API access available (not publicly announced)
  - Allows export of dashboard data, responses, and citation frequency
  - Detailed API guide to be provided separately

### Company Background

  - Scrunch AI launched from beta in February
  - Grown from 3 to 12 team members in recent months
  - Serving diverse clients from large enterprises to small businesses
