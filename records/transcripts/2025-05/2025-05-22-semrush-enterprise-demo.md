## Key Topics

### AIO Product Overview

  - Uses Keyword Magic Tool to generate prompts and questions
  - Tracks AI platform mentions and sentiment across multiple models
  - No API access currently available, limiting integration possibilities
  - Still in early stages, with focus on adding markets/languages

### Enterprise SEO Platform Capabilities

  - Provides detailed traffic and keyword performance analysis
  - Offers content optimization tools with AI-powered recommendations
  - Includes technical site auditing and page speed analysis
  - Allows for A/B testing and Google Analytics integration

### Pricing Structure

  - Based on tiered system (Bronze, Silver, Gold, Platinum)
  - Customized pricing depends on:
      - Number of client projects
      - Keywords tracked per client
      - Technical crawling needs
      - Service token usage for various features

  - growthx.ai team to review demo recording internally
  - Semrush to provide detailed pricing proposal
