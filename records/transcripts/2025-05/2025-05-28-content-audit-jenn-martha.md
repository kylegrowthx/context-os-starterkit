## Key Topics

### SEO-Driven Content Strategies

  - Utilize meta fields for SEO keywords, allowing more human-readable titles in content
  - Consider using table of contents for better Google snippets
  - Balance SEO requirements with readability and narrative flow

### AI Content Generation Challenges

  - Hallucinated stats and links are major time-sinks for editors
  - AI tends to overuse bullet points and create "bullet point sandwiches"
  - Tone consistency issues, especially for clients requiring a balance (e.g., playful yet professional)
  - Outlines often need manual refinement before content generation

### Improving Workflow Instructions

  - Current polishing instructions are insufficient; need significant expansion
  - New standard: extensive, detailed instructions with specific do's and don'ts
  - Example shared from AIMBIT project: multi-page document with comprehensive guidelines
  - Instructions should include negative examples ("instead of this, use that")

### Content Structure and Readability

  - Aim for "geometric" variety in content structure to improve readability
  - Avoid walls of text and overly skeletal content
  - Use varied paragraph lengths and occasional single-sentence paragraphs for emphasis

### Client-Specific Calibration

  - Implement a new process for kicking off clients with intense calibration
  - Create detailed writing guides for each client based on feedback and collaboration
  - Use tools like Perplexity AI to generate initial writing guides from collected feedback
