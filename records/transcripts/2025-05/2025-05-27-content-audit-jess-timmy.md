## Key Topics

### Headline Structure

  - LLMs tend to overuse colon structure in headlines (70%+ of content)
  - Recommendation: Avoid this format, even editing pre-given headlines if necessary
  - Test readability by speaking headlines aloud; if it's awkward, revise

### Introduction Optimization

  - Aim for 2-3 paragraphs, 2-5 sentences each
  - Include valuable statistics and keyword-rich content for SEO snippets
  - Balance between brevity and providing a compelling overview
  - Use intro as a "mini recap" to entice readers and qualify their interest

### Statistics and Source Management

  - Request client-approved sources during kickoff meetings
  - Two-step process for finding statistics:
    1.  Ask LLM for reputable sources on the topic
    2.  Request statistics only from those verified sources
  - Future workflow: Develop a "statistics populator" tool for easier insertion

### Content Focus and Depth

  - Prioritize outcomes over feature lists
  - Example: "Self-directed learning activity grew by 40%" instead of listing LMS features
  - Expand on crucial topics (e.g., "cold start problem" in LMS) that align with article premise
  - Use internal tools (e.g., ChatGPT panels) to identify and emphasize key pain points

### Client-Specific Considerations

  - Adapt content style to client preferences (e.g., bullet point density)
  - Leverage client case studies and data when available
  - Balance between general best practices and client-specific requirements
