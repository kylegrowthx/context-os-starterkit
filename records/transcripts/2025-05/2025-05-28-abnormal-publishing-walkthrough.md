## Overview

- Content publishing process involves creating entries in Craft CMS, with specific steps for blogs vs. glossary pages
- Hidden posts can be published immediately, but live edits should be toggled off during updates
- SEO elements like title tags, meta descriptions, and image alt text need to be manually added
- New functionality allows for adding FAQ accordions to glossary pages

---

## Key Topics

### CMS Overview and Blog Post Creation

  - Craft CMS used for content management
  - Blog post creation involves:
      - Setting title, slug, meta description
      - Toggling "hide post" for non-public content
      - Adding content blocks (text, headings, images)
      - Setting categories and authors
      - Configuring SEO settings

### Glossary Page Management

  - Similar process to blog posts, with additional considerations:
      - Show/hide summary option affects index page display
      - FAQ schema deprecated but still used for search engines
      - New accordion functionality for adding FAQs to page content
  - Some glossary pages currently empty, need to be filled out

### Image Handling

  - No strict file size requirements for in-content images
  - Naming convention: blog name or shortened version with underscore and number
  - Open graph images automatically pulled for social sharing

### SEO and Metadata

  - Title tags can be customized separate from H1
  - Meta descriptions pulled from summary field by default
  - FAQ schema still used despite Google deprecation

### Publishing Workflow

  - Toggle off live status during edits to prevent partial updates
  - Can use draft functionality or direct edits
  - Immediate publishing possible for hidden posts

### Interlinking Strategy

  - Prioritize links to main Abnormal Security site over Intelligence site
  - Open to adding new glossary terms and filling out empty pages
