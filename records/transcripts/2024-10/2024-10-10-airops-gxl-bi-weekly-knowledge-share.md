## Overview

- AirOps introduced new image-related features: hero images, open graph images, and interspersed article images
- New "Read from Grid" step allows filtering and fetching data from grids within workflows
- Custom variables in brand kits enable more flexible and specific content creation
- GrowthX Labs provided feedback on permissioning, workflow optimization, and training needs

---

## Key Topics

### New Image Features

  - Three new image templates: hero images, open graph images, and interspersed article images
  - Uses Getty (premium) and Unsplash (budget-friendly) APIs for image sourcing
  - Human review step for image selection, with multi-select option for interspersed images
  - Open graph images use brand colors and logos fetched via BrandFetch API

### Read from Grid Feature

  - New step allowing filtering and reading data from grids within workflows
  - Useful for fact-checking, content optimization, and dynamic content creation
  - Enables more complex workflows and data-driven content generation

### Custom Variables in Brand Kits

  - Allows creation of specific variables beyond standard brand kit fields
  - Enables more tailored content creation for different content types (e.g., listicles)
  - Improves workflow customization and flexibility

### Q4 Product Roadmap

  - Focus on improving output quality and reducing time to value
  - Planned improvements to Studio UX, workspace organization, and quality evaluations
  - Investing in AI features like co-pilot for workflow improvements

### User Feedback and Requests

  - Need for more granular permissioning and access control
  - Desire for infographic creation capabilities
  - Interest in YouTube video embedding within articles
  - Request for lead capture functionality similar to chatbase.co
  - Need for best practices in prompt engineering and model selection
