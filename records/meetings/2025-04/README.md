## File Naming Convention

Files follow the pattern: `YYYY-MM-DD-descriptive-name.md`

Examples:
- `2025-04-04-growthx-directors-meeting.md`
- `2025-04-10-kickoff-vapi-growthx.md`
- `2025-04-24-vapi-growthx-weekly-sync.md`

Rules:
- Lowercase
- Hyphens separate words
- Special characters removed
- Truncated to ~60 characters
- Automatic suffix for duplicate filenames
