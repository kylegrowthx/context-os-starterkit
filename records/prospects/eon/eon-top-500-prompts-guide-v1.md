## How This Guide Is Organized

This guide follows the CheckThat prompt taxonomy — a six-axis classification system that treats prompts as research instruments, not random questions. Every prompt is designed to probe a specific dimension of how AI perceives Eon, classified by:

1. **Buyer Stage** — Where the buyer is in their journey
2. **Question Type** — The structural category (16 types)
3. **Intent** — The cognitive action (Learn, Explore, Compare, Validate, Act)
4. **Measurement Purpose** — What the prompt reveals (Presence, Perception, or both)
5. **Context Modifiers** — Industry, company size, role, use case, tech stack
6. **Perception Attribute** — Which of the 6 perception attributes it probes (Capability, Usability, Value, Trust, Support, Innovation)

### Prompt Tiers

Each prompt is assigned a priority tier based on buyer realism, commercial intent, measurement value, competitive differentiation, and volume proxy.

| Tier | Priority | Action |
|------|----------|--------|
| **T1** | Critical | Track weekly. Immediate content action on gaps. |
| **T2** | High | Track weekly. Queue content work. |
| **T3** | Medium | Track bi-weekly. Monitor. |
| **T4** | Lower | Monthly monitoring. |

## Appendix E: How to Use This Guide

### Getting Started (First Week)

1. **Start with T1 prompts only** (~150 prompts). These are the highest-value evaluation-stage prompts.
2. **Run across 4 engines:** ChatGPT, Perplexity, Claude, Gemini.
3. **Minimum 30 runs per prompt** for statistical significance.
4. **Record:** Was Eon mentioned? Position? Sentiment? Who else appeared? What sources were cited?
5. **Calculate baseline:** Presence Rate, Position Score, Cross-Engine Coverage.

### Weekly Tracking

1. Run all T1 prompts weekly.
2. Run T2 prompts bi-weekly.
3. Run T3-T4 prompts monthly.
4. Flag: drops >10% week-over-week, new competitor appearances, incorrect information.

### Gap-to-Content Loop

| What the data shows | Content action |
|---|---|
| Eon doesn't appear in "best cloud backup for [X]" | Create definitive page: "Cloud Backup for [X]" with Eon positioning |
| Eon doesn't appear in "[Competitor] alternatives" | Build comparison/alternatives content; ensure differentiation is clear |
| AI says incorrect pricing | Update pricing page, add schema markup, refresh FAQ |
| AI doesn't mention CBPM | Publish authoritative content defining CBPM; build Wikipedia presence |
| AI doesn't mention data activation | Create content around backup data lakes; publish case studies |
| Eon mentioned but not cited | Improve content structure — lead with answers, add schema markup |
| Competitor cited from third-party source | Create your version; build relationships with that source |
| Low perception on a specific attribute | Create content targeting that attribute's prompt patterns |

### Alert Thresholds

- Presence Rate drops >10% week-over-week on T1 prompts
- Sentiment turns negative for 2 consecutive runs
- Incorrect pricing or product info appears in any response
- A competitor newly appears on prompts where they weren't before
- Eon disappears from an engine where it was previously visible
