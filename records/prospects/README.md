# Prospects

Companies GrowthX has engaged with that are not currently active clients. Includes past prospects, churned accounts, and deal room materials.

---

## How to Use

1. **One folder per company.** Named by slug (e.g., `simple-life/`).
2. **Search first.** Use grep/search to find companies by name.
3. **Load individually.** Read specific company files — never bulk-load.
