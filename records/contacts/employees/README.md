# Employees

GrowthX team directory. Sourced from Slack workspace, enriched over time.

---

## How to Use

1. **One file per employee.** Named `firstname-lastname-v1.md`.
2. **Search first.** Use grep/search to find people by name, title, or department.
3. **Load individually.** Read specific employee files — never bulk-load.
4. **Keep profiles current.** Update after role changes, departures, or onboarding.

## Employee File Structure

Each employee file includes:

- **Metadata** — name, title, department, Slack ID, tags, last_updated
- **Role** — current title, department, reporting context
- **Background** — how they joined, previous roles if known
- **Working Style** — communication preferences, strengths, notes from interactions
- **Timeline** — start date, role changes, key milestones
