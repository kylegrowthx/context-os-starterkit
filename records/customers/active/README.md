# Active Customers

Current GrowthX client accounts. Each folder contains client profiles, operating manuals, strategy documents, and meeting transcripts.

---

## How to Use

1. **One folder per client.** Named by slug (e.g., `airbyte/`, `vercel/`).
2. **Search first.** Use grep/search to find clients by name, ME, or topic.
3. **Load individually.** Read specific client files — never bulk-load.
4. **Notion is source of truth.** Client Workspaces DB in Notion has the latest contract details, health scores, and pod assignments. These files capture context for AI access.

## Notion Source

- [Client Workspaces DB](https://www.notion.so/growthxlabs/14f2ba60bc74808799a9f49603d6882c) — master database with all client workspaces, contracts, MEs, status
- [Client DB](https://www.notion.so/growthxlabs/1f22ba60bc7480bcac19efcbce270c93) — health scores, pod assignments, weekly updates
