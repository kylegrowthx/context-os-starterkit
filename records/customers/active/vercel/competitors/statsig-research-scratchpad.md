## 1. COMPANY OVERVIEW — Founding Story, History, Key Milestones

### Founding and Early Development

**Founders & Team Formation**

- Vijaye Raji founded Statsig in February 2021 with a core team of 7 engineers joining on day one
- Raji's background: Former Microsoft (10 years, Windows Application Framework, SQL Server, Visual Studio). Then Facebook (10 years as VP Gaming, VP Entertainment, led mobile ad product)
- At Facebook, Raji worked with internal tools including Gatekeeper (feature flagging), Deltoid (experimentation/metrics), and Quick Experiments
- Vijaye's core insight: "Most teams are using tools that were far behind what we took for granted at Facebook"
- Statsig's mission: Democratize Facebook's internal feature management and experimentation tools for companies of all sizes

## 5. PRODUCT & FEATURE ANALYSIS

### Core Product Stack

Statsig is an integrated platform with five core pillars:

**1. Feature Flags & Configuration**
- Runtime feature toggling for safe rollouts
- Percentage-based rollouts with gradual ramp-up
- Targeting and segmentation (user cohorts, custom attributes)
- Feature gate rules evaluation at sub-millisecond latency
- Free at any scale (charged via events only)

**2. Experimentation & A/B Testing**
- Full experimental framework with statistical rigor
- **Sequential Testing:** Safe peeking with adjusted confidence intervals (mSPRT method)
- **Multivariate Testing:** Test multiple variants and interactions
- **Holdout groups:** Measure long-term effects against holdout cohorts
- **Multi-metric analysis:** Scorecard approach to weigh conflicting metrics
- **Power analysis calculator:** Estimate experiment duration upfront
- Pulse analysis for continuous experiment monitoring
- Integration with metrics and analytics

**3. Product Analytics**
- Event tracking and metrics
- Metrics Explorer with background query support
- User journey analysis
- Custom event properties
- Warehouse-native analytics (run in Snowflake, BigQuery, etc.)
- Event-level user flow visualization

**4. Session Replay**
- Record user sessions with full interaction history
- Link replays to experiments, flags, and metrics
- Event-level granularity (every action annotated)
- Conditional recording triggers (only record when criteria met)
- Cost-effective selective recording

**5. Warehouse Native (2023+)**
- Run experimentation directly in customer data warehouse
- Supported: Snowflake, BigQuery, Databricks, Redshift
- Uses warehouse compute for all statistical analysis
- Preserves data privacy (no data egress)
- Transparent statistics engine (can inspect SQL, methodology, intermediate datasets)

### Feature Comparison: Statsig vs Vercel

| Feature | Statsig | Vercel Flags SDK | Vercel Toolbar | Gap Assessment |
|---------|---------|-----------------|------------------|----------------|
| **Feature Flags** | Full-featured gates, targeting, rollouts | Basic flags via Edge Config + providers | Override flags in toolbar | Statsig: Full platform |
| **Experimentation** | Advanced A/B testing, sequential testing, multivariate | Not built-in (use provider) | Not built-in | **Statsig: Strong advantage** |
| **Analytics** | Native product analytics + warehouse native | Not included | Not included | **Statsig: Unique advantage** |
| **Session Replay** | Linked to experiments and flags | Not included | Not included | **Statsig: Advantage** |
| **Integration** | Native with Vercel Edge Config | Part of ecosystem | Integrates with providers | Parity on Vercel |
| **Statistical Methods** | Sequential testing, safe peeking, holdouts | Provider-dependent | N/A | **Statsig: Advanced** |
| **SDKs** | 20+ (JS, Python, mobile, server) | Vercel-specific for Edge Config | Browser-based | Statsig: Broader SDK coverage |
| **Pricing Model** | Free flags, events-based analytics | Edge Config free tier, provider pricing | Free toolbar | Statsig: Transparent metering |
| **Development Experience** | Dashboard + code | Code-first (Flags as Code) | UI management | Different paradigms |

### Key Differentiators vs Competitors

**1. Statistical Rigor**
- Sequential testing with safe peeking (only experimentation platform with this capability)
- mSPRT methodology — allows peeking without inflating false positives
- Power analysis calculator for experiment duration estimation
- Holdout groups for long-term effect measurement

**2. Warehouse Native**
- Only experimentation platform allowing analysis in customer's data warehouse
- Preserves data sovereignty and privacy
- Transparent statistical engine

**3. All-in-One Platform**

**4. Vercel Integration**

## 6. PRICING & PACKAGING

### Statsig Pricing Tiers

**Developer Tier (Free)**
- 2M metered events per month
- Unlimited feature flags (free at any scale)
- 50K session replays
- Includes experimentation, analytics, session replay
- Free tier charges nothing — no limits on flags
- Use case: Individual developers, small teams, hobbyist projects

**Pro Tier**
- Baseline: $150/month
- Included: 5M metered events per month
- Overage: $0.05 per 1K additional events
- Cost grows with scale and event volume
- Ideal for: Growing startups, mid-market teams

**Enterprise Tier**
- Custom pricing with volume discounts
- Triggers: Typically at $20M+ monthly events
- Discount: 50%+ volume discounts available
- Includes: Advanced support, SLA, advanced governance
- Ideal for: Large enterprises, AI platforms

### Pricing Philosophy

- **Free flags:** Feature flags are always free, regardless of scale or volume
- **Events-based metering:** Charged only for analytics events (gate checks are free)
- **Transparent scaling:** Usage dashboard shows real-time event volume and costs
- **Growth-friendly:** Free tier removes barrier to adoption; Pro tier allows self-serve scaling

### Pricing Comparison with Competitors

| Tier | Statsig | LaunchDarkly | PostHog | Vercel |
|------|---------|--------------|---------|--------|
| **Free** | 2M events/mo, unlimited flags | Limited (10K events/mo) | Limited (1M events/mo) | Free (Edge Config) |
| **Pro** | $150 + usage | $300-1000+/mo | $200-2000+/mo | Provider pricing |
| **Enterprise** | Custom (50%+ discount) | Custom (high) | Custom | Provider + Edge Config |

**Key Insight:** Statsig's free flags model is unique — LaunchDarkly, PostHog, and Vercel all charge for flags in some form.
