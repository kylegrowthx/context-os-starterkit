## 5. Product & Feature Analysis

### Core Platform Capabilities

**Object Storage:**
- S3-compatible API (supports most common S3 actions)
- Native B2 API for full feature access
- Always-hot storage at cold storage prices (no tiering)
- 11 nines (99.999999999%) data durability guarantee
- File upload sizes: up to 5GB via CLI/Native API, up to 10TB for large files
- Supports parallel uploads of large files (5 parts to 10,000 parts per file)

**Storage Tiers:**
- B2 Standard (hot storage, $6/TB/month)
- B2 Overdrive (high-performance, $15/TB minimum, terabit-speed throughput, unlimited egress)
- No cold/archive tiers (unlike AWS S3 Glacier, Cloudflare R2)

**Data Ingress:**
- File uploads: 10-30% faster for small files (1MB or less) vs AWS S3
- 40% faster for large files (e.g., 1TB VM backups) in benchmarks
- Free uploads (no egress costs for uploads)

**Data Egress:**
- Free egress up to 3x average monthly storage
- $0.01/GB beyond 3x
- **Unlimited free egress** to partner CDNs (Cloudflare, Fastly, bunny.net, CoreWeave, etc.)
- **B2 Overdrive:** Unlimited egress included
- No per-request API charges (unlike AWS S3, GCP, Azure)

**Storage Features:**
- Object Lock (immutability for compliance; Compliance Mode, Legal Hold, Default Bucket Retention)
- Server-side encryption (Backblaze-managed keys SSE-B2, or customer-managed SSE-C)
- HTTPS only (non-secure connections rejected)
- IPv6 support
- Automatic incomplete upload cleanup via lifecycle rules

**Developer Experience:**
- Web UI, CLI, REST API
- Official SDKs: Python (b2sdk), Java
- AWS SDK compatibility (JavaScript V3, Python, .NET)
- 250+ community repositories in 10+ languages (C#, Go, Ruby, Elixir, etc.)
- CORS support for browser-based access
- Postman API collection for testing
