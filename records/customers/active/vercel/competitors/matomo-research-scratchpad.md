## 1. COMPANY OVERVIEW & FOUNDING

### Founding Story & History
- **Founded:** 2007 by Matthieu Aubry (New Zealand-based)
- **Original Name:** Piwik (rebranded to Matomo in 2017)
- **Founding Context:** Aubry created an open-source web analytics project to provide privacy-first alternative to Google Analytics
- **Meaning:** "Matomo" means "honesty" in Japanese, reflecting values around privacy and data ownership
- **Key Milestone:** 1.4M+ websites across 190+ countries using Matomo as of 2025
- **10-Year Bootstrap:** Unusual company path—raised first funding 10 years after founding (2017), reflecting early bootstrap strategy

## 5. PRODUCT & FEATURE ANALYSIS

### Core Analytics Features
- **Page Analytics:** Pageviews, unique visitors, bounce rate, time on page
- **Heatmaps:** Click heatmaps, scroll tracking, element-level interaction visualization
- **Session Recordings:** Full visitor session playback with mouse movements, scrolls, form fills
- **Form Analytics:** Form completion tracking, field-level abandonment, time to fill
- **Goals & Funnels:** Conversion tracking, multi-step funnel visualization
- **E-Commerce Tracking:** Product views, cart abandonment, purchase tracking, revenue reporting

### Advanced Features (Premium/Plugins)
- **Tag Manager:** Built-in tag management (similar to GTM but first-party owned)
- **Cohort Analysis:** Premium plugin for user segmentation and behavior analysis
- **Advanced Funnels:** Premium plugin with multi-step path analysis
- **A/B Testing:** Experimental design with traffic splitting
- **Segments:** Custom audience segments with sharing and copying (roadmap item)
- **Custom Dimensions:** Product-level and visitor-level custom data
- **Activity Log:** Audit trail of all admin and user changes
- **API Access:** Full data retrieval via REST API for custom integrations

### Deployment & Hosting
- **Matomo Cloud:** Managed SaaS with EU data residency (Germany)
- **On-Premise/Self-Hosted:** Full open-source option with complete code access
- **Dual Model:** Unique positioning allowing both maximal control (self-hosted) and minimal maintenance (cloud)

### Privacy & Compliance Features
- **Cookie-Less Tracking:** IP anonymization (configurable 1-3 bytes), privacy by default
- **GDPR Compliance:** Right to erasure, right to access, consent management built-in
- **CCPA Compliance:** Cookie consent, opt-out mechanisms, data export functionality
- **Data Residency:** EU-based data centers for cloud deployments
- **Data Retention:** Configurable from 1 month to indefinite
- **No Third-Party Data Sharing:** All data stays in Matomo (no Google, Facebook pixel overlap)

## 6. PRICING & PACKAGING

### Matomo Cloud SaaS Pricing
- **Starter:** $23/month (up to 50K hits/month, 30 websites)
- **Business:** Tiered pricing based on monthly hits (50K → 100K → 500K → 1M+)
- **Premium Plugins:** Additional cost for heatmaps, A/B testing, cohort analysis, advanced funnels
- **Overage Charges:** €2.20 per 5,000 hits beyond monthly allowance
- **Enterprise Cloud:** $16,900/month+ for 100M+ monthly hits with custom terms

### Matomo On-Premise (Self-Hosted)
- **Free Open-Source:** Zero license cost for core platform
- **Infrastructure Costs:** Self-funded server costs, backups, security maintenance
- **Premium Support Plans:** Available for self-hosted deployments (response time ~1 business day)
- **Premium Plugins:** Annual subscription for advanced features on self-hosted

### Vercel Comparison
| Dimension | Matomo | Vercel Web Analytics |
|-----------|--------|----------------------|
| **Free Tier** | Self-hosted only; Cloud free trial | Included in Hobby plan |
| **Entry Price** | $23/month (cloud) | $20/user/month (Pro) |
| **Pricing Model** | Hit-based (with flat monthly) | User-based + consumption |
| **Self-Hosting** | Yes (free core) | No |
| **Enterprise Path** | Custom/contact sales | Custom/contact sales |

**Key Insight:** Matomo's free self-hosted option and lower cloud entry point ($23) vs Vercel's $20/user create different economics for SMBs. Matomo wins on cost for single-user teams; Vercel's per-user model penalizes multi-person teams.

## KEY INSIGHTS FOR COMPETITOR BRIEF

### Matomo's Core Competitive Advantages
1. **Data Ownership:** Complete control via self-hosting or EU-based cloud
2. **Open Source:** 20.7K GitHub stars, 200K+ community, transparent codebase
3. **Privacy First:** GDPR/CCPA native, no data selling, no third-party overlap
