## 9. Competitive Positioning (50+ sources)

### Amplitude vs. Direct Competitors

**Amplitude vs. Mixpanel:**
- Mixpanel: Event-driven, focuses on user analytics
- Amplitude: Cohort-first, product analytics leader
- Mixpanel switched to event-based pricing (Feb 2025) — more predictable
- Mixpanel free plan: 20M events/month (more generous)
- Amplitude stronger in experimentation; Mixpanel in pure analytics

**Amplitude vs. PostHog:**
- PostHog: Open-source, engineer-focused, includes SQL queries in free tier
- Amplitude: Enterprise-focused, proprietary, cloud-only
- PostHog: Lower cost of entry, self-hosting option
- Amplitude: More polished UX, stronger integrations
- PostHog winning with technical product teams; Amplitude with growth/marketing

**Amplitude vs. Heap:**
- Heap: Strong session replay and autocapture
- Amplitude: Stronger experimentation and behavioral analytics
- Heap: Better for friction/UX insights
- Amplitude: Better for growth and retention optimization

**Amplitude vs. Google Analytics 4:**
- Google Analytics: Broader web analytics, free tier, integration with ecosystem
- Amplitude: Specialized product analytics, advanced segmentation, deeper insights
- Market segmentation: GA4 for basic web metrics; Amplitude for product team insights

### Amplitude vs. Vercel Web Analytics & Speed Insights

**Direct Comparison:**
- **Amplitude:** Product analytics, behavioral data, experimentation platform
- **Vercel Web Analytics:** Basic web analytics (page views, referrers, geography)
- **Vercel Speed Insights:** Core Web Vitals monitoring and performance metrics

**Market Positioning:**
- Amplitude and Vercel serve different audiences:
  - Amplitude: Product managers, growth teams, data teams
  - Vercel Web Analytics: Developers using Vercel platform
  - Vercel Speed Insights: DevOps/infrastructure engineers, performance specialists

**Overlap Areas:**
- Both track user behavior
- Both provide dashboards
- Both enable data-driven decision-making

**Differentiation:**
- Amplitude: Deep user segmentation, cohort analysis, experimentation, predictive analytics
- Vercel Web Analytics: Privacy-focused, minimal integrations, lightweight, platform-native
- Vercel Speed Insights: Performance metrics, Core Web Vitals, technical performance only

**For Vercel:**
- Amplitude is adjacent, not direct competitor
- Vercel customers may use both (Amplitude for product insights, Vercel for performance)
- Potential partnership/integration opportunity
- Minimal cannibalization risk
