# Churned Customers

Former GrowthX client accounts. These companies were tagged as `customer` in HubSpot but are no longer active engagements.

---

## How to Use

1. **One file per client.** Flat structure — no subfolders.
2. **Search first.** Use grep/search to find clients by name or domain.
3. **Reactivation candidates.** If a churned client re-engages, move their profile to the parent `/customers/` directory and update status to `active`.

## Source

- **HubSpot** — all companies with `lifecycle_stage = customer` that are not in the active client list or prospects.

## Count

**48 churned client accounts.**
