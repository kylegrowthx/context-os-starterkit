## How This Was Built

Started with 200+ sources extracted from project documents and research, then curated to 100 based on:
1. Signal-to-noise ratio (high value per piece)
2. Actionability (specific, not just conceptual)
3. Relevance to our stage ($10M-$50M ARR, hybrid services+tech)
4. Update frequency (for time-sensitive topics)
