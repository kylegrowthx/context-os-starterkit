## Audience Analysis

| Question | Answer |
|----------|--------|
| WHO is learning this? | GrowthX product & delivery teams building a page scoring system for Content OS and CheckThat |
| WHAT do they care about vs noise? | Actionable scoring factors with evidence, not speculation. Weighted factors that can be computed programmatically. |
| WHY do they need this? | To build a per-page health/quality/discoverability score that Content OS uses to prioritize work and CheckThat uses to measure AI visibility at the page level |

## Research Categories

### 1. Page Health
- Core Web Vitals (LCP, INP, CLS)
- Page speed metrics
- Indexing status and crawlability
- Mobile-friendliness
- HTTPS/security
- Structured data validity
- HTML validity / accessibility
- Server response times / TTFB
- Redirect chains
- Broken links / 404s
- Image optimization
- JavaScript rendering

### 2. Page Discoverability
- Internal linking structure
- External backlink profile
- XML sitemap inclusion
- Canonical tags
- Hreflang implementation
- robots.txt directives
- Indexing API submissions
- Social sharing metadata (OG tags)
- AI engine crawlability
- Schema markup for discovery

### 3. SEO Factors
- On-page optimization (title, meta, headers)
- Content relevance and depth
- Keyword optimization
- Topical authority signals
- Backlink quality and quantity
- Domain authority signals
- Fresh content signals
- User intent matching
- SERP feature eligibility

### 4. AEO Factors
- AI citation likelihood
- Semantic completeness
- Answer-first formatting
- Multi-modal content signals
- Citation-ready content structure
- AI Overview selection criteria
- Cross-engine visibility
- Source authority for LLMs
- Factual verifiability

### 5. Page Quality
- E-E-A-T signals (experience, expertise, authoritativeness, trust)
- Content depth and comprehensiveness
- Originality / unique value
- Readability and clarity
- Author attribution
- Editorial standards
- Factual accuracy
- Visual quality
- Content freshness

### 6. Page Engagement
- Predicted dwell time
- Scroll depth indicators
- Internal navigation paths
- CTR optimization (titles/meta)
- Content format variety
- Interactive elements
- Reading experience (layout, typography)
- Exit rate predictors
- Conversion path indicators

### 7. Additional Factors
- Brand signals at page level
- Entity optimization
- Accessibility (WCAG)
- Security signals
- Privacy compliance
- Internationalization
- Performance budget adherence
