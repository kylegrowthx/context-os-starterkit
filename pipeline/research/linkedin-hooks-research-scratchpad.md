## Key Frameworks Discovered

### 1. The 5 Core Hook Formulas
**Source:** Superpen, multiple creators
1. Contrarian Hook — Challenge the norm
2. Relatable Pain Hook — Call out shared frustration
3. Confession Hook — Raw personal story
4. List/Teaser Hook — Promise numbered value
5. Big Claim/Outcome Hook — Lead with results

### 2. SLAY Framework (Lara Acosta)
- **S**tory: Personal insight
- **L**esson: Identify the problem
- **A**ctionable: Lists for clarity
- **Y**ou: Address reader directly

### 3. Jasmin Alić's 7 Rules
1. One-liner hooks only
2. First 3 lines = entire post outline
3. Numbered listicles
4. Never start with "stop/don't/quit"
5. Add signposting (social proof)
6. Short, choppy copy
7. End with question or P.S.

### 4. Dwell Time Optimization
**Source:** LinkedIn Engineering
- 61+ seconds dwell = 15.6% engagement
- 0-3 seconds dwell = 1.2% engagement
- First 60-90 minutes = 70% of reach

### 5. Algorithm Priorities (2025-2026)
- Comments weighted 8x more than likes
- Saves 10x more valuable than likes
- Native content outperforms links 1.9x
- Engagement bait is suppressed

## Patterns Observed

### What Works
1. One-line hooks with specific numbers
2. Personal pronouns ("I", "My", "How I")
3. Vulnerability paired with credibility
4. Native formats (carousels, text, video)
5. Questions that invite real discussion
6. Consistency over 90+ days
7. Responding to every comment in first hour

### What Fails
1. Multi-line hooks (20% worse performance)
2. Engagement bait ("Like if you agree!")
3. External links in post body
4. AI-generated content (30% less reach)
5. Off-topic/inconsistent posting
6. Starting with "Stop", "Don't", "Quit"
7. Vague, generic openings
