# Hooks and offers

All marketing starts with the offer. This skill teaches how to frame offers as invitations or transformations, write memos before building, and create hooks that stop the scroll.

## Trigger phrases

- Creating campaigns, funnels, or event pages
- Building landing pages or marketing assets
- Designing offers or writing hooks

## What it does

Starts with the WAFM rule — Write a Freaking Memo before anything else. Then walks through four steps: write the memo, frame the transformation (current state to desired state), simplify until someone in an Uber after two drinks can understand it, and name your system (the "4-1-2 Open House Formula" approach).

Includes the Why/What/How/Now validation framework, volume-based testing methodology, and high-converting hook patterns. Applies the same visualization, falsifiability, and uniqueness tests from the copywriting skill to offers.
