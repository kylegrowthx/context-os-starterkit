## Publishing Process

1. **Log in with team@growthxlabs.com** gmail into [Sanity and add yourself as an author](https://www.sanity.io/@ohRRuwym4/studio/l3d2swt9tl7g1s72dmpbu84t/default/structure/post)

2. **Choose a topic** in Airtable (or add anything you add directly in assignment):  
   https://airtable.com/appb8TtXzukwdUINA/tbloJmnYIe1w2NfPc/viwjJ6jI8WHgq8Tqb?blocks=hide

3. **Write the content** using one of our pipelines:  
   https://atlas.growthx.ai/accounts/101/workspaces/0bfe3a5c-0509-49f1-9c4d-45fa22f976ce/projects/82886841-6fb5-4bda-86a5-da2736aa02ee/task_pipelines

4. **Come back 20 to 30min later** and adjust the final article as much as you want + double check everything

5. **Copy the data into Sanity** and publish it manually (our current setup doesn't have Sanity auto-publishing yet)

6. **Update Airtable** with the Assignment status
