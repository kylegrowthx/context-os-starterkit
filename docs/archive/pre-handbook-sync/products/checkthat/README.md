# CheckThat Documentation

CheckThat is the open AI visibility index for B2B. This directory contains all strategic documentation for the product.

---

## Quick Summary

CheckThat tracks how AI engines answer B2B buyer questions. We map categories, brands, prompts, and responses—then make it all public.

**The play:** Build the open AI visibility index, give away more value than competitors charge for, monetize just enough to cover costs.

**The moat:** Historical data that can't be backfilled + editorial depth competitors can't replicate.
