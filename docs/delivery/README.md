## The Delivery Model

```
Sales → Strategy Sprint (8 weeks) → Growth Execution (ongoing)
          ↓                              ↓
     Convert to GE                   Pods (EM + Editors)
```

**Three teams:**
- **Sales**: Qualify and close new business
- **Strategy Sprint**: Run 8-week sprints, convert to Growth Execution
- **Delivery**: Run Growth Execution in pods
