# EPD (Engineering, Product & Design)

This directory contains documentation for GrowthX's EPD organization—how we build product, structure teams, and operate.

---

## Quick Summary

EPD has 3 lanes:

1. **Product Lane:** Design Engineers and Full-stack engineers building ContentOS (Atlas) and CheckThat.ai
2. **Client Ops Eng Lane:** Forward-deployed Marketing & AI Engineers creating custom workflows and covering client needs
3. **AI Infra Lane:** Team in charge of the internal AI framework Output
